# Liar Module

이 디렉토리는 `Discord Game Bot` 안에서 Discord 채널 기반 `라이어게임` 모듈을 담당한다. 웹 대시보드는 쓰지 않고, 생성된 채널 안에서 버튼, 일반 메시지, `!투표` prefix, `/제시어` ephemeral 응답만으로 한 판을 끝내는 구조다.

## 현재 입력 체계

- slash command: `/liar-create`, `/제시어`
- 버튼: `참가`, `나가기`, `시작`, `투표 시작`, `강제 집계`, `종료`
- prefix: `!투표 @대상`
- 일반 메시지:
  - 설명 단계의 한 줄 설명
  - 라이어 추리 단계의 최종 정답 입력

## 안정화/UX

- 로비 `10분` 무활동 시 자동 종료
- 설명 `45초`, 투표 `45초`, 추리 `30초`
- 토론은 인원수 기반 자동 타이머로 진행 후 자동 투표 시작
- 단계 종료 전 경고 메시지 발송
- 상태 메시지에 단계, 진행도, 참가자, 남은 시간을 함께 표시

## 현재 범위

- `4~8인`
- 서버당 동시 게임 `1개`
- 제시어 카테고리 기반 출제
- `/제시어` 기반 개인 정보 전달
- 채널 안에서 로비, 설명, 토론, 투표, 추리, 종료까지 진행
- 상태 저장은 `in-memory`

## 운영 메모

- 이 모듈은 Discord `Message Content Intent` 가 켜져 있어야 `!투표` 와 일반 메시지 입력을 처리할 수 있다.
- 재시작 시 진행 중 게임은 복구되지 않는다.

## 주요 문서

- `RULE.md`: 라이어게임 단일 기준 규칙 문서
- `PLAN.md`: 확장 계획과 후속 작업 메모

현재 루트 봇 런타임은 `mafia/`와 `liar/`를 함께 실행하며, 루트 `build`와 `test`에도 `liar/` 소스가 포함된다.
