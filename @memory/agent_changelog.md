# Agent Changelog

## Prompt Semantic Versioning Policy

- 형식: `v[Major].[Minor].[Patch]-[AgentName]`
- `Major`: 핵심 목적, 메모리 구조, 프롬프트 아키텍처 변경
- `Minor`: 행동 지침/워크플로/도구 사용 정책 추가
- `Patch`: 문구 수정, 오탈자, 제약 보강
